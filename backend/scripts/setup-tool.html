HubertStudios License Manager — Setup Tool

Runs entirely in this browser tab using the Web Crypto API. Nothing is sent anywhere — no network requests are made by this page. Safe to use on your phone. You can save this file locally and open it offline if you prefer.

This replaces the scripts/generate-keypair.js and scripts/hash-password.js Node scripts from the README, since you're setting this up from the Cloudflare dashboard without a CLI.

1. Generate the ECDSA signing keypair

The private key goes into the Worker as the PRIVATE_KEY secret. The public key goes into your Java plugin's License.java (the publicKeyPem you pass to LicenseGate.Config.builder()).

2. Generate the admin password hash

Pick a strong admin password (12+ characters). This page derives a PBKDF2-SHA256 hash (210,000 iterations) and a random salt — these are what you store as ADMIN_PASSWORD_HASH and ADMIN_PASSWORD_SALT. The plaintext password is never sent anywhere; only the resulting hash/salt are meant to be pasted into Cloudflare. Save the plaintext password itself in your own password manager — it cannot be recovered from the hash.

3. Generate SESSION_SECRET

A long random string used to sign session cookies and verification codes.

Where these go in the Cloudflare dashboard: open your Worker → Settings → Variables and Bindings → Add → choose type Secret (not "Text") for every value generated here, so it's encrypted at rest and hidden after saving.
